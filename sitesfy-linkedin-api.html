LinkedIn API 方案对比

四种接入方式全面评估 + 针对桌游出海场景的落地建议

版本 v1.0 日期 2026-03-23 场景 Sitesfy · 义乌桌游公司 Outbound
四种方案速览

Proxycurl

推荐 · 最稳定

第三方 LinkedIn 数据 API，专为产品集成设计。个人资料、公司信息、搜索、联系方式一站覆盖，有 SLA 保证。

$0.01–0.05 / 请求
按量付费 Company Search API ✓ 推荐

RapidAPI LinkedIn

测试阶段可用

多个供应商（LinkedIn Data API、Fresh LinkedIn Profile Data 等），价格便宜，适合 MVP 早期测试和概念验证。

更低，按套餐
多供应商 稳定性参差 ○ 测试

官方 LinkedIn API

门槛极高

需申请 Marketing Developer Platform 或成为合作伙伴。审核周期长，需要一定规模公司。最稳定但最难拿到。

免费（但难拿）
需审核 周期长 ✗ 暂缓

PhantomBuster / Apify

高风险，不推荐

模拟浏览器自动化，非真正 API。批量获取数据但账号随时可能被封，不稳定，Sitesfy 产品层不适合依赖。

月费制

LinkedIn 之外，这些数据源更直接：欧洲桌游零售商本身在 LinkedIn 上密度不高，专业渠道覆盖更全。

Faire.com

欧美最大批发平台

桌游零售商密度极高，有公开 API。可直接找到买家，比 LinkedIn 更精准。

Apollo.io

MVP 已规划接入

按 ICP 条件拉取公司 + 联系人，比 Proxycurl 便宜且覆盖更广。Sitesfy MVP 第一期主数据源。

ICP 筛选邮箱直达

SPIEL 展商名单

德国 Essen 桌游展

每年公开，欧洲零售商最集中的一手数据源。德国市场验证的天然起点。

免费德国市场高质量
🎯
Proxycurl 最有价值的时机：找到目标公司名之后，用 Proxycurl 补充「决策人」的 LinkedIn 信息（职位、邮箱线索），配合开发信个性化。Apollo 有时候漏人，Proxycurl 补这一层最准。
